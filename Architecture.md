# AI Sentinel Architecture

## System Overview

AI Sentinel is an AI-powered security system designed to prevent NEET question paper leaks by monitoring activities, detecting suspicious patterns, and generating alerts.

## Architecture Flow

Data Sources  
↓  
Data Collection Layer  
(Exam logs, access records, digital activity data)

↓  

AI Processing Layer  
(Machine Learning + Deep Learning models)

↓  

Suspicious Activity Detection  
(Detect unusual access, abnormal behaviour, and possible threats)

↓  

Risk Analysis Engine  
(Classify risk levels: Low / Medium / High)

↓  

Alert Generation System  
(Send alerts to examination authorities)

↓  

Final Action  
(Investigation, access control, or clearance)

## Technologies Used

- Artificial Intelligence
- Machine Learning
- Deep Learning
- Data Analysis

## Objective

To create a secure and transparent examination system by detecting and preventing possible question paper leaks using AI.
