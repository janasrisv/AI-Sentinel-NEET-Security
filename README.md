
# AI Sentinel - NEET Security

AI-powered solution for preventing NEET question paper leaks.

## Problem Statement
NEET question paper leaks affect fairness, student trust, and exam integrity.

## Solution
AI Sentinel is an AI-based security system that monitors suspicious activities and detects possible threats before they become leaks.

## Features
- AI-based anomaly detection
- Suspicious activity monitoring
- Alert generation
- Secure exam process

## Technology Used
- Artificial Intelligence
- Machine Learning
- Data Analysis
